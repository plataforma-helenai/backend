🎯 Diferencial #1

Onboarding inclusivo que adapta tudo automaticamente

No primeiro acesso, o aluno responde 4 perguntas simples sobre seu perfil. A plataforma ativa automaticamente os recursos certos sozinha.

  • ✓ Daltonismo → ativa paleta Color Blind
  • ✓ Dislexia → ativa fonte OpenDyslexic
  • ✓ Visual → ativa contraste + ledor + fonte grande
  • ✓ Auditiva → ativa intérprete LIBRAS
  • ✓ Intelectual → ativa Linguagem Simples + ledor
🤖 Diferencial #2

IA "Helen" como assistente pedagógica

Na hora da prova, o aluno tem uma toolbar com 4 botões poderosos integrados ao enunciado:

  • 💡 Simplificar — reescreve em Linguagem Simples lado a lado
  • 🔑 Palavras-chave — destaca termos com tooltip
  • 📖 Pedir uma dica — sem revelar a resposta
  • 🔊 Ouvir questão — TTS com destaque do trecho
💙 Diferencial #3

Feedback humanizado, nunca punitivo

Aplicamos princípios DUA em cada mensagem da plataforma. A Helen explica cada questão errada com tom acolhedor:

  • ✗ "Você errou" → ✓ "Vamos revisar juntas? 💪"
  • ✗ "Nota baixa" → ✓ "Você está progredindo!"
  • ✗ "Tempo esgotado" → ✓ "Sem pressa, continue quando quiser"

Cronômetro é sempre opcional. Pressão não educa.

⚔ Diferencial #4

Desafios temáticos com ranking próprio

Professores e alunos podem criar "períodos de estudo" agrupando vários simulados, com ranking, pontuação e gamificação.

  • 🎯 Mínimo 2 simulados · pontuação = soma da % de acerto
  • 📅 Prazo opcional · pode ser contínuo
  • 👥 Vinculado a turmas ou aberto a todos
  • 🏆 Ranking próprio + posição no global do aluno
⭐ Principal diferencial

Métricas de inclusão como produto

O dashboard administrativo tem uma seção dedicada chamada "💙 Impacto e Inclusão" mostrando o que mais nos importa:

  • 📊 Donut chart — alunos PcD por tipo de deficiência
  • ♿ Barras animadas — % usando cada recurso de acessibilidade
  • 📈 Linguagem Simples: 47% · Ledor: 34% · LIBRAS: 11%
  • 💙 Não falamos de inclusão. Medimos.
♿ Diferencial #6

Acessibilidade nativa em cada tela

Barra de acessibilidade fixa em todas as 36 telas, mesmo antes do login. 6 toggles sempre acessíveis:

  • ◐ Alto contraste — paleta otimizada
  • Aa Fonte OpenDyslexic — para dislexia
  • A+ Tamanho de fonte — 3 níveis
  • 👁 Color Blind — paleta acessível
  • 🔊 Ledor TTS — texto para voz com controle
  • 🤟 LIBRAS — janela com intérprete virtual
